# Побитови операции
Прилагат се върху един бит или набор от повече отделни битове на двоични числа.

 - Побитово И (**&**)    

|        |           |   
|--------|-----------|   
| a      | 1001 0110 |   
| b      | 1010 1100 |
| a&b    | 1000 0100 |


 - Побитово ИЗКЛЮЧВАЩО ИЛИ  (**^**)

|        |           |
|--------|-----------|
| a      | 1001 0110 |
| b      | 1010 1100 |
| a&b    | 0011 1010 |

 - Побитово отместване (**<<**) (**>>**)


|        |           |
|--------|-----------|
| a      | 1001 0110 |
| a << 2 | 0101 1000 |
| a >> 2 | 0010 0101 |

- Побитовa негация (**~**) 


|    |           |
|----|-----------|
| a  | 1001 0110 |
| ~a | 0110 1001 |




## Задачи

**1.** Да се напише функция, която с побитови операции проверява дали число е четно

**Пример:**

```c++
isEven(8) ---> true
isEven(7) ---> false
```

**2.** Да се напише функция, която с побитови операции повдига 2 на степен k.

**Пример:**

```c++
powerOfTwo(8) ---> 256
```

**3.** Да се напише функция, връща стойността на бит на дадена позиция(брои се от дясно наляво).

**Пример:**
```c++
а = 150 // 1001 0110
checkBit(а, 3) ---> 0
checkBit(а, 4) ---> 1
```

**4.** Да се напише функция, която занулява бит на дадена позиция.

**Пример:**

```c++
а = 150 // 1001 0110
clearBit(a, 2) ---> 146 // 1001 0010
clearBit(a, 3) ---> 150 // 1001 0110
```

**5.** Да се напише функция, която прави бит единица по подадена позиция.

**Пример:**

```c++
а = 150 // 1001 0110
makeBitOne(a, 5) ---> 182 // 1011 0110
makeBitOne(a, 7) ---> 150 // 1001 0110
```


**6.** Да се напише функция, която обръща бит на дадена позиция.

**Пример:**

```c++
а = 150 // 1001 0110
toggleBit(a, 3) ---> 158 // 1001 1110
toggleBit(a, 2) ---> 146 // 1001 0010
toggleBit(a, 7) ---> 18  // 0001 0010
```


**7.** Да се напише функция, която променя бит по подадена позиция на подадена стойност 0 или 1.

**Пример:**

```c++
а = 150 // 1001 0110
setBit(a, 0, 0) ---> 150 // 1001 0110
setBit(a, 6, 1) ---> 214 // 1101 0010
setBit(a, 1, 0) ---> 148 // 1001 0100
```

**8.** Да се напише функция, която приема цяло число и обръща най-десния бит в него, който е единица.

**Пример:**

```c++
а = 150 // 1001 0110
flipMostRigth1(a) ---> 148 // 1001 0100
```

**8.** Да се напише функция, която приема цяло число и обръща най-левия бит в него, който е нула.

**Пример:**

```c++
а = 150 // 1001 0110
flipMostRigth1(a) ---> 214 // 1101 0110
```

**9.** Да се напише функция, която приема 2 цели числа a и b и разменя стойностите им без допълнителна променлива с побитови операции.

**10.** Да се напише функция, която приема 2 цели числа a и b и проверява дали битовете на a са пермутация на битовете на b.

**Пример:**

```c++
а = 7 // 0111
b = 11 // 1011
c = 8 // 1000
arePermutations(a, b) ---> true
arePermutations(a, c) ---> false
```

**11.** Да се напише функция, която брои единиците в двоичния запис на число.

**Пример:**

```c++
countOnes(8) ---> 1
countOnes(11) ---> 3
```

## Бонус задачи

Напишете функция, която приема такъв масив и връща кое е това число.
(Подсказка: използвайте побитови операции, за да постигнете линейно решение)

*Вход: [9 18 9 12 18 15 12], Изход: 15*


**Задача 2:** Напишете функция, която приема масив(разглеждаме го като множество) и отпечатва всички негови подмножества(без значение е редът на отпечатване).

*Вход: [1, 2, 3], Изход: [], [1], [2], [3], [1,2], [2,3], [1,3], [1,2,3]*

*Вход: [5, 3] Изход: [], [5], [3], [5, 3]*

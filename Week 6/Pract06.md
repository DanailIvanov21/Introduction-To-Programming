#  Двумерни масиви (матрици) в C++

##  Определение

**Двумерният масив (матрица)** е структура от данни, съставена от елементи **от един и същи тип**, подредени в **редове и колони**.

---

##  Синтаксис

```cpp
<тип> <име_на_масив>[<брой_редове>][<брой_колони>];
```

---

##  Семантика

В паметта се заделят последователни клетки с общ брой:
`брой_редове * брой_колони`
Общият размер = `(брой елементи) * (размер на един елемент)`.

---

##  Пример

```cpp
int matrix[2][3];
```

➡️ Създава матрица с 2 реда и 3 колони.

---

## ⚙ Инициализация

###  Начин 1:

```cpp
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
```

###  Начин 2:

```cpp
int matrix[2][3] = {1, 2, 3, 4, 5, 6};
```

###  Начин 3:

```cpp
int matrix[2][3];
matrix[0][0] = 1;
matrix[0][1] = 2;
matrix[0][2] = 3;
// и т.н.
```

---

##  Недопустими операции

```cpp
int matrix1[2][3], matrix2[2][3] = {{1, 2, 3}, {4, 5, 6}};

matrix1 = matrix2; // ❌ Недопустимо: не може да се присвояват цели масиви
```

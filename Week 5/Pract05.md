```c++
#include <iostream>
double f()
{
     double num1 = 4;
     double num2 = 4 / 7;
     return (num2-- = ++num1 + 2);
```

# Задачи, максималният размер да е 1000

**1.** Да се напише функция, която приема масив от цели числа и размера му, и връща дали е сортиран.

Вход:
```c++
[5, 7, 4, 9]
```
Изход:
```c++
false
```
Вход:
```c++
[1, 2, 3, 4]
```

Изход:
```c++
true
```

**2.**  Да се напише функция, която приема масив от цели числа и размера му, и връща средната стойност на елементите му и числото, което е най-близо до тази средна стойност.

**Пример:**

Вход:
```c++
[5 7 4 9]
```

Изход:
```c++
6.25
7
```

**3.** Напишете функция, която приема масив от цели числа, размера му и число N, и обръща първите N елемента на масива.

Вход:
```c++
[1, -4, 4, -5, -9, 2, 10]
4
```

Изход:
```c++
[-5, 4, -4, 1, -9, 2, 10]
```

**4.**  Напишете функция, която приема масив от цели числа и индекс и премахва числото, което се намира на този индекс.

Вход:

```
[33 1 23 8 54 5 1 6] 3
```

Изход:

```
[33 1 23 54 5 1 6]
```
